Разбор Теста по Физике: Оптика и Атомная Физика

Вариант 1

1. Гармонические колебания

  • Условие: x = Acos(ωt). При фазе φ = π/3 смещение x = 3 см. Найти амплитуду A.
  • Решение: Подставляем значения в x = Acos(φ): 3 = A · cos(π/3). Так как cos(π/3) = 1/2, получаем 3 = A · (1/2). Отсюда A = 3 · 2 = 6 см.
  • Ответ: 6 см

2. Максимальная длина звуковой волны

  • Условие: Диапазон слышимости 20 - 20'000 Гц. Скорость звука Vз = 340 м/с. Найти λmax.
  • Решение: Длина волны λ = V / ν. Максимальная длина волны соответствует минимальной частоте (νmin = 20 Гц). λmax = 340 м/с / 20 Гц = 17 м.
  • Ответ: 17 м

3. Разность фаз

  • Условие: λ = 0,6 мкм, разность хода Δl = 0,6 мкм. Найти разность фаз Δφ.
  • Решение: Δφ = (2π / λ) · Δl = (2π / 0,6) · 0,6 = 2π. По условию записываем как 2п.
  • Ответ: 2п

4. Дифракция Френеля на диске

  • Условие: Непрозрачный диск закрывает две первые зоны Френеля. I0 — интенсивность падающей волны. Найти интенсивность I в центре.
  • Решение: В центре дифракционной картины за диском наблюдается пятно Пуассона-Араго. Если закрыты первые k зон, амплитуда в центре A ≈ Ak+1/2. Здесь k=2, A ≈ A3/2. Так как A3 ≈ A1, а амплитуда от всей волны A0 = A1/2, то A ≈ A0. Следовательно, I ≈ I0.
  • Ответ: Б) I = I₀

5. Дифракция на щели

  • Условие: λ = 410 нм, b = 20,5 мкм, θ = 30°. Найти порядок минимума m.
  • Решение: Условие минимума: b · sin(θ) = m · λ. Отсюда m = (b · sin(θ)) / λ = (20,5·10⁻⁶ · sin(30°)) / (410·10⁻⁹) = (20,5·10⁻⁶ · 0,5) / (410·10⁻⁹) = 25.
  • Ответ: Г) m = 25

6. Период дифракционной решетки

  • Условие: N₀ = 625 штрихов на L = 1 мм. Найти период d.
  • Решение: d = L / N₀ = 1 мм / 625 = 0,0016 мм = 1,6 мкм.
  • Ответ: Б) d = 1,6 мкм

7. Закон Малюса

  • Условие: Интенсивность уменьшилась в 4 раза. Найти угол α между поляризатором и анализатором.
  • Решение: I₁ = Iест / 2. I₂ = I₁ · cos²(α) = (Iест / 2) · cos²(α). По условию I₂ = Iест / 4. Iест / 4 = (Iест / 2) · cos²(α) → cos²(α) = 1/2 → α = 45°.
  • Ответ: B) 45°

8. Закон Стефана-Больцмана

  • Условие: Температура T увеличилась в 2 раза. Как изменилась площадь под графиком?
  • Решение: Площадь под графиком (интегральная светимость) R ∝ T⁴. Если T₂ = 2T₁, то R₂ ∝ (2T₁)⁴ = 16T₁⁴. Площадь увеличится в 16 раз.
  • Ответ: Увеличится в 16 раз

9. Перевод эВ в Джоули

  • Условие: Работа выхода Авых = 2,5 эВ.
  • Решение: Авых = 2,5 · (1,6 · 10⁻¹⁹ Дж) = 4,0 · 10⁻¹⁹ Дж.
  • Ответ: Г) 4·10⁻¹⁹ Дж

10. Волна де Бройля

  • Условие: m = 10⁻⁹ кг, λд = 6,63·10⁻³⁰ м. Во сколько раз v < c?
  • Решение: v = h / (m · λд) = (6,63·10⁻³⁴) / (10⁻⁹ · 6,63·10⁻³⁰) = 10⁵ м/с. Отношение c / v = (3·10⁸) / (10⁵) = 3000.
  • Ответ: Д) В 3000 раз

11. Радиусы орбит в атоме Бора

  • Условие: Найти отношение r₄ / r₂ (n=4 к n=2).
  • Решение: rn ∝ n². Отношение r₄ / r₂ = 4² / 2² = 16 / 4 = 4.
  • Ответ: E) 4

12. Максимумы вероятности в потенциальной яме

  • Условие: Электрон в состоянии n=3. Найти координаты максимумов.
  • Решение: Максимумы плотности вероятности (пучности) для n=3 находятся в точках x = a/6, x = a/2, x = 5a/6.
  • Ответ: В) x₁=a/6; x₂=a/2; x₃=5a/6

13. Вероятность нахождения в потенциальной яме

  • Условие: n=3, яма шириной l. Вероятность в интервале (l/3, 2l/3).
  • Решение: График плотности вероятности для n=3 имеет 3 симметричных "горба" равной площади. Интервал (l/3, 2l/3) точно соответствует центральному "горбу". Вероятность равна 1/3.
  • Ответ: Б) 1/3

14. Радиоактивные распады

  • Условие: Превращение ²³⁵₉₂U → ²⁰⁷₈₂Pb. Найти число α- и β⁻-распадов.
  • Решение: α-распады: x = (235 - 207) / 4 = 7. β⁻-распады: 92 = 82 + 2·7 - y → 92 = 96 - y → y = 4.
  • Ответ: B) 7α и 4β

15. Период полураспада

  • Условие: T = 20 мин, m₀ = 40 г. Найти массу через 1 час (60 мин).
  • Решение: Пройдет n = 60/20 = 3 периода. Масса уменьшится в 2³ = 8 раз. m = 40 г / 8 = 5 г.
  • Ответ: Д) 5 г

16. Ядерная реакция

  • Условие: ¹⁰B + n → ⁷Li + X. Определить Х и число нейтронов.
  • Решение: ¹⁰₅B + ¹₀n → ⁷₃Li + ᴬZX. A: 10+1 = 7+A → A=4. Z: 5+0 = 3+Z → Z=2. Х — это гелий ⁴₂He. Число нейтронов N = 4-2=2.
  • Ответ: Элемент Х — гелий (He), число нейтронов — 2

Основные формулы

Закон Стефана-Больцмана:
R = σT⁴
Закон смещения Вина:
λmax · T = b
Разность фаз:
Δφ = (2π / λ) · Δl
Закон радиоактивного распада:
N(t) = N₀ · (1/2)t/T
Альфа-распад (α-распад):
AZX → A-4Z-2Y + 42He
Бета-распад (β⁻-распад):
AZX → AZ+1Y + 0-1e + ν̅e
Волна де Бройля:
λ = h / p = h / (mv)
Закон Малюса:
I = I₀ · cos²(α)
